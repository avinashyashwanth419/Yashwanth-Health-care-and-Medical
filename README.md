# sns-hackathon
Health and Medical
problem statement:
Healthcare decisions are complex due to multiple factors: patient history, genetics, medications, and evolving medical literature.
Clinicians need evidence-based personalized treatment plans for safer and more effective care.
description
This system analyzes individual patient characteristics, including:
Age:
Medical history / chronic conditions:
Current medications:
Genetic risk factors:
It integrates medical knowledge and drug interactions to generate:
Personalized treatment recommendations
Warnings for potential drug interactions
Predicted risk score for patient outcomes
Monitoring and follow-up suggestions
features:::
Patient Input: Age, conditions, medications, genetic risk
Treatment Recommendation: Evidence-based therapies from a knowledge base
Drug Interaction Check: Flags risky drug combinations
Risk Prediction: Calculates risk score using patient data
Interactive Frontend:
Responsive UI using Bootstrap
Displays treatments, warnings, and risk visualization chart
Real-time Demo: Frontend communicates with backend API via FastA
